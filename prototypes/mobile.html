Decisions baked in: a fixed bottom tab bar (Library / Log / Search / Settings), no header search on Library (the Search tab is the entry point), the Library split stacks vertically, and dialogs become bottom sheets. No account/avatar chrome — sign-out lives in Settings. Switch shelf tabs or open the sheet to feel the interactions. This is the working reference for the React build.
